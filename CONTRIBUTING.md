# Contributing to Knowledge Tier Framework for AI Agents

Thank you for your interest in contributing to the Knowledge Tier Framework for AI Agents! This project is maintained by Theios Research Institute, Inc. and we welcome contributions from the community.

## How to Contribute

### Reporting Issues

- Use GitHub Issues to report bugs or suggest features
- Check existing issues before creating a new one
- Include your operating system and shell (bash/zsh)
- Provide steps to reproduce any bugs

### Suggesting Enhancements

We're particularly interested in:

- Additional tier templates for specific use cases
- Integration with other security frameworks
- Industry-specific tier configurations (healthcare, finance, legal)
- Documentation improvements
- Cross-platform compatibility (Windows support)

### Submitting Pull Requests

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Make your changes
4. Test thoroughly on your system
5. Commit with clear messages (`git commit -m "Add: description of change"`)
6. Push to your fork (`git push origin feature/your-feature`)
7. Open a Pull Request

### Code Style

- Shell scripts require **bash** (use `#!/bin/bash` shebang)
- Bash-specific features like `[[ ]]` and `=~` are acceptable
- Use clear variable names
- Include comments for complex logic
- Follow existing formatting patterns
- JSON files should be properly formatted

### Testing

Before submitting:

- Test on macOS and/or Linux
- Test with both bash and zsh
- Verify the install script works on a clean system
- Test tier initialization for all four tiers
- Verify templates copy correctly

## Code of Conduct

- Be respectful and inclusive
- Focus on constructive feedback
- Help others learn and grow
